# ProxyScraper
Simple Proxy Scraper written in Python.

**Features:**
- Duplicates Remover
- Port Remover
- Previus Proxies
- Protocol Remover
- MultiThreaded
- Well Optimized
- Fully Customizable
- Open Soruce

**Usage:**
- Change config.json
- Add Links to socks4.txt/socks5.txt/http.txt
- python3 ProxyScraper.py
