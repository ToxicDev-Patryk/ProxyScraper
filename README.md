# ProxyScraper
Simple Proxy Scraper written in Python.

**Features:**
- Duplicates remover
- MultiThreaded
- Protocol remover
- previous proxies
- Clean IP:Port (Removed everything except ip:port)
- Remove local (Removes Local Addresses)
- Highly Customizable

**Usage:**
- Change config.json
- Add Links to scrape
- python3 ProxyScraper.py
